# TP-JO
Le but de ce TP réalisé en binôme était de prédire l'ensemble des médailles susceptible d'être gagné par chaque pays lors de cette édition des Jeux Olympiques (2024), en fonction des précédentes éditions qu'il y a déjà eu, répertorié dans l'ensemble des bases de données pour observer des corrélation importantes entre les vainqueurs et les éditions.

Les données remonté depuis 1896, comme vous pouvez vous en doutez il fallait réaliser plusieurs tâches avant de pouvoir prédire les médailles gagné, car des sports n'existe plus, des pays comme la Yougoslavie non plus. D'autres pays eux ont changé de nom comme l'URSS ect ...
Donc c'est après un long travail de traitement, nettoyage et de visualisation de donnée que nous avons pu créer notre algorithme que je vous laisse découvrir
